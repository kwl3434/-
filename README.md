# RoutinePlay

현실에서 각종 루틴을 수행하면 나만의 캐릭터가 성장한다.<br>
가상의 캐릭터와 함께 실제로 성장해나가는 자신의 모습을 보고 성취감을 느낄 수 있는 어플

#### 차별점
어플시장에 많은 루틴 어플들이 존재하지만 모두 수행한 루틴을 기록하는 것에 그친다.<br>
이를 확장시켜 루틴을 수행함에 따라 가상의 캐릭터가 성장하는 모습을 확인 가능하도록 한다.<br>
